# HostelHub Figma Screens Explained

## 1. Login Screen
- **Why:** Lets students securely access the app.
- **Features:** App logo, email & password fields, login button.

## 2. Dashboard/Home
- **Why:** Quick overview and easy navigation.
- **Features:** Welcome message, buttons to report/view issues.

## 3. Issue Reporting Form
- **Why:** Simple way to submit new complaints.
- **Features:** Dropdown (issue type), text field (description), submit button.

## 4. Issue List & Status
- **Why:** Lets students track their complaints and see updates.
- **Features:** List of issues, colored status badges (Open, In Progress, Resolved).

**Each screen solves a real user need and keeps the app simple.**
